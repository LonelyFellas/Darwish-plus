## 描述
- 该库提供了一系列的判断函数，用于判断传入的参数是否为某种类型
- 该库提供了一系列的判断函数，不仅仅是类型判断还有其他功能的判断比如: `dev`, `promise`, `jsx`, `browser`, `email`等等

## `index.ts`导出的所有的组件
```jsx
export * from "./isTypings"; //判断类型
export { default as isBlanks } from "./isBlanks"; //判断参数是否为 true
export { default as isEmailValid } from "./isEmailValid"; // 判断邮箱是否合法
export { default as isBrowser } from "./isBrowser"; // 判断是否为浏览器环境
export { default as isDev } from "./isDev"; // 判断是否为开发环境
export { default as isJSX } from "./isJSX"; // 判断是否为JSX组件
export { default as isPromise } from "./isPromise"; // 判断是否Promise函数
```


### `isTypings.ts`导出的所有的组件
<span style={{color: 'red'}}>注意：这里的判断也都是带着ts类型的判断</span>
- `isDate` - 判断是否为日期
- `isObject` - 判断是否为对象
- `isSet` - 判断是否为Set
- `isMap` - 判断是否为Map
- `isNull` - 判断是否为Null
- `isBigint` - 判断是否为大数
- `isArray` - 判断是否为数组
- `isSymbol` - 判断是否为Symbol
- `isString` - 判断是否为字符串
- `isFunction` - 判断是否为函数
- `isBoolean` - 判断是否为布尔值
- `isNumber` - 判断是否为数字
- `isUndef` - 判断是否为undefined

## 使用示例
```tsx
import {isBoolean, isPromise, isDev} from "@darwish/utils-is";

export default () => {
    const promise = new Promise((resolve, reject) => {
        resolve('success');
    });

    const ans: boolean | string = "false"

    if (isBoolean(ans)) {
      console.log('ans is boolean'); // 类型 ans is boolean
    } else {
      console.log('ans is string'); // 类型 ans is string
    }

    if (isPromise(promise)) {
      console.log('promise is promise'); // 类型 promise is promise
    } else {
      console.log('promise is not promise'); // 类型 promise is not promise
    }

    return (
      <div>
          <h1>utils-is</h1>
          {isDev() ? <h2>开发环境</h2> : <h2>生产环境</h2>}
      </div>
    );
}
```
